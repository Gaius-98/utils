# utils
gaius发布的工具
# 安装
```
npm i gaius-utils
```
# 使用
1. 全部引入
  ```
  import GaiusUtils from 'gaius-utils'
  import 'gaius-utils/lib/style.css'
  createApp(App).use(GaiusUtils)
  ```
2. 部分引入
  ```
  import {xxx} from 'gaius-utils'
  ```
